# BACNext

BACNext | bilibili API Collect Next | 本项目与 BAC 没有从属关系

## 快速开始

您可以使用 Apifox 文档快速完成调试和文档查看。

API 文档公开发布地址: https://bacnext.apifox.cn

## 运作方式

本项目全匿名化贡献，我们使用 `Apifox` 完成对 API 文档的编写和发布，并将定期同步 OpenAPI 文件到本仓库。

在未来如果可能，我们将以 MarkDown 的形式同步 Apifox 公开页的文档到本仓库。

## 如何贡献

对于 Api 完善与修改方面，您可联系 `bacnext<at>protonmail<dot>com` 申请加入我们，我们会邀请您加入我们的 Apifox 组织，相关更改全部在 Apifox 完成。

对于 自动化同步等 基础设施方面，您可直接对本仓库发起 PR，或联系 `bacnext<at>protonmail<dot>com`。
